# Repetición Máxima - Frontend

Aplicación web para registrar entrenamientos y analizar la progresión del 1RM.

## Instalación
1. `npm install`

## Comandos
- `npm run dev`: Inicia el servidor de desarrollo (Vite) en `http://localhost:3000`.
- `npm run build`: Construye la aplicación para producción.

## Configuración
Vite esta configurado con un proxy para redirigir las llamadas a `/api` hacia `http://localhost:5000` (Backend).
